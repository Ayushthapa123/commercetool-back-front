## OpenAPI Specification of Janus APIs

The [OpenAPI Specification](https://swagger.io/specification/) defines the contract design of HTTP APIs. The yaml file 
"commercetoolapi" provides specification of Janus APIs such as products, CRUD carts, etc. 

## How to Run Janus API specification:

Any Http API client that supports yaml file can execute APIs. For instance [Client Scalar API](https://client.scalar.com/workspace/default/request/default)
is one of the API client can be used. 

## How to Update Janus API specification:

Here are some of the applications that can be use to update API specification:
1. [API-Fiddle](https://api-fiddle.com/)
2. [Swagger](https://swagger.io/)
3. [OpenAPI Editor 42 Crunch VS Code Extension](https://marketplace.visualstudio.com/items?itemName=42Crunch.vscode-openapi)
